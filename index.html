Before We Begin The JS Quiz
Rule 1. You only have 20 seconds per each question.
Rule 2. Once you pick an answer, you can't go back and change it.
Rule 3. When the time runs out for a question, you can't select an answer if it wasn't selected before.
Rule 4. Exiting before the quiz is finished will also end with a failure.
Rule 5. Once the quiz is finished, you will get points for only the correct answers.
Rule 6. Good luck and have fun!
